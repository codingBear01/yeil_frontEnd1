내부스타일이 적용된 태그입니다.

내부스타일은 해당 스타일이 작성된 웹문서 전체에 적용되는 내용이기 때문에 인라인과 달리 1회용이 아닌 여러번 사용이 가능하다.

단점이 있다면 해당 내부스타일에 적용된 내용을 확인하기 위해서 헤드에 작성한 스타일을 눈으로 확인해야한다는 번거로움이 있다.

이미 내부 스타일이 적용된 태그에 다시 인라인 스타일을 추가로 적용시킬 수도 있다. 이 경우에는 내부 스타일과 인라인스타일 모두가 적용이 된다.

스타일값이 중복되는 경우 (내부스타일에서 적용했던 스타일을 인라인에서 다른값으로 적용시키는 경우) 나중에 작성된 스타일이 최종적으로 적용된다.
